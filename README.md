# kafka-use-cases
Just son simple but useful use case using Kafka

### Start dev env
```
docker-compose up
```

### Landoop UI
    http://localhost:3030


### SERVICES PORTS
    9092 : Kafka Broker
    8081 : Schema Registry
    8082 : Kafka REST Proxy
    8083 : Kafka Connect Distributed
    2181 : ZooKeeper
    3030 : Web Server
